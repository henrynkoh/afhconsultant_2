# AFH Compliance Architect - Step-by-Step Tutorial

Welcome! This tutorial will guide you through using the AFH Compliance Architect platform to master AFH compliance consulting.

## Tutorial Overview

**Duration**: 30-45 minutes  
**Prerequisites**: None  
**Goal**: Learn to navigate and effectively use all platform features

---

## Part 1: Getting Started (5 minutes)

### Step 1: Access the Platform
1. Open your web browser
2. Navigate to `http://localhost:3000` (or your deployed URL)
3. You'll see the homepage with 5 module cards

### Step 2: Understand the Homepage
- **Module Cards**: Each card represents a curriculum module
- **Icons**: Visual indicators for each topic area
- **WAC References**: Shows which regulations each module covers
- **R.A.R. Method Box**: Explains the consultant's logic framework

### Step 3: Your First Module
Click on **"Setting Up Your Home"** to begin your first module.

## Part 3: Using the Question Bank (15 minutes)

### Step 1: Understanding Question Structure
Scroll down to "Top 100 Preparation Questions" section.

Each question shows:
- **Question Number**: Q1, Q2, etc.
- **Category Badge**: Physical Plant & Safety, etc.
- **WAC Badge**: Regulation reference (if applicable)
- **Question Text**: The actual question
- **Chevron Icon**: Click to reveal answer

### Step 2: Revealing Answers
**Try This**:
1. Read question Q1: "What is the maximum allowable temperature for water at the tap?"
2. Try to answer it yourself first
3. Click the chevron (â–¼) icon
4. Notice how the answer expands with:
   - Direct answer
   - WAC citation
   - Detailed explanation

### Step 3: Category Filtering
**Try This**:
1. Look for the dropdown menu above the questions
2. Click it to see available categories
3. Select "Physical Plant & Safety"
4. Notice how only questions in that category are shown
5. Select "All Categories" to see everything again

### Step 4: Study Technique
**Recommended Approach**:
1. **Read** the question
2. **Think** about your answer
3. **Reveal** the answer
4. **Compare** your answer with the provided answer
5. **Study** the explanation and WAC citation
6. **Note** any areas you need to review

## Part 6: Study Workflow (5 minutes)

### Recommended Daily Workflow

**Morning Session (30 minutes)**:
1. Review one curriculum phase
2. Read all content carefully
3. Take notes on key points
4. Review WAC citations

**Afternoon Session (30 minutes)**:
1. Practice with 10-15 questions

3. Create flashcards for key facts

## Part 7: Tips for Success

### Tip 1: Use Category Filters
- Focus on one category at a time
- Master each category before moving on
- Review weak categories more frequently

### Tip 2: Understand, Don't Memorize
- Focus on the "why" behind each answer
- Understand the logic and risk assessment
- This helps with similar but different questions

### Tip 3: WAC Citations Matter
- Note which WAC applies to which topic
- Understand the regulation structure
- This helps when consulting with providers

### Tip 4: Regular Review
- Don't just read once
- Review questions multiple times
- Space out your reviews (spaced repetition)

### Tip 5: Practice Explaining
- Try explaining answers to someone else
- Teaching reinforces learning
- If you can explain it, you understand it

## Troubleshooting

### Question: I can't see the questions
**Answer**: Scroll down on the module page. Questions are below the curriculum phases.

### Question: The answer won't expand
**Answer**: Make sure you're clicking the chevron icon, not just the question text.

### Question: I want to print questions
**Answer**: Use your browser's print function. Answers will print in their expanded state if you expand them first.

### Question: Can I use this on mobile?
**Answer**: Yes! The platform is fully responsive and works on all devices.
